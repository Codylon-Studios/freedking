WORKFLOW:(mit cmd option)
Neuer branch auf Basis der develop branch erstellen:
git branch -c <branchname>
Dann auf git pushen:
git push origin <branchname>
Im git graph sollte neben eurem branch ein label mit origin erschienen sein
(Graphischer Pfad ohne cmd: rechtsklick auf den commit auf den develop zeigt-> create branch
dann zum pushen nochmal rechtsklick -> push branch; jetzt solte auch origin auftauchen)
Der aktuelle Branch ist der der fett gekennzeichnet ist; wenn nicht (zur Sicherheit)
Am Anfang git checkout <branchname> ausführen

etwas programmieren
git add .
git commit (dann taucht eine Datei name commit_msg auf; euren commit reinschreiben und datei schließen)
git push origin <branchname> (akutalisiert mit Änderungen auf github)
